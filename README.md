# Turtle Drawing Controller (Python)

A Python Turtle graphics project that allows users to control a turtle using keyboard inputs to draw on the screen. Event-driven programming and interactive input handling.

# ▶️ How to Run

1. Make sure you have Python installed on your computer.

2. Download or clone this repository.

3. Ensure the file is in your folder:

    main.py

4. Open a terminal (or your preferred Python IDE).

5. Navigate to the folder containing the file.

6. Run the script using:

    python main.py

7. A Turtle graphics window will open. Use the keyboard controls to move and draw.

🧠 Controls

- W → Move forward  
- S → Move backward  
- A → Turn left  
- D → Turn right  
- C → Clear drawing and reset  

🧠 Concepts Practiced

- Event-driven programming using keyboard listeners
- Handling user input with `screen.onkey()`
- Creating and organizing multiple functions
- Controlling object movement and direction
- Managing object state (position and heading)
- Using Turtle graphics for interactive applications
- Separating input handling from movement logic
